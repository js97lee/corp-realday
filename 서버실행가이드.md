# 서버 실행 가이드 (초보자용)

## 📌 전체 흐름 이해하기

이 프로젝트는 **2개의 서버**가 필요합니다:

1. **프론트엔드 서버** (React) - 포트 3000
   - 사용자가 보는 웹페이지
   - 이미 실행 중일 수 있음

2. **백엔드 서버** (Express) - 포트 5000
   - 데이터를 처리하는 서버
   - 지금 설치하고 실행해야 함

---

## 🚀 단계별 실행 방법

### 1단계: 백엔드 서버 설치하기

**터미널을 열고** (새 터미널 창을 열어주세요):

```bash
# 1. 프로젝트 폴더로 이동
cd /Users/jisulee/A.홈페이지

# 2. server 폴더로 이동
cd server

# 3. 백엔드 의존성 설치 (이게 중요!)
npm install
```

**설치가 완료되면:**
- `node_modules` 폴더가 생성됩니다
- "added XXX packages" 같은 메시지가 나옵니다
- 몇 분 정도 걸릴 수 있습니다

---

### 2단계: 백엔드 서버 실행하기

**같은 터미널에서:**

```bash
npm run dev
```

**성공하면 다음과 같은 메시지가 나옵니다:**
```
서버가 포트 5000에서 실행 중입니다.
http://localhost:5000
```

**이 터미널 창은 계속 열어두세요!** 
- 서버가 실행 중이라는 뜻입니다
- 닫으면 서버가 꺼집니다

---

### 3단계: 프론트엔드 서버 확인하기

**다른 터미널 창을 열고:**

```bash
# 프로젝트 폴더로 이동
cd /Users/jisulee/A.홈페이지

# 프론트엔드 서버 실행 (이미 실행 중이면 생략)
npm run dev
```

**프론트엔드 서버가 실행되면:**
- 브라우저가 자동으로 열리거나
- `http://localhost:3000` 주소가 표시됩니다

---

## ✅ 확인 방법

### 백엔드 서버가 제대로 실행되었는지 확인:

브라우저에서 직접 접속해보세요:
- `http://localhost:5000/api/contacts` (GET 요청)

또는 터미널에서:
```bash
curl http://localhost:5000/api/contacts
```

### 프론트엔드와 백엔드가 연결되었는지 확인:

1. 브라우저에서 `http://localhost:3000/admin` 접속
2. 로그인 시도:
   - 이메일: `admin@real-day.com`
   - 비밀번호: `admin123`
3. 성공하면 서버 연동이 잘 된 것입니다!

## 🔧 문제 해결

### 문제 1: "npm: command not found"
→ Node.js가 설치되지 않았습니다
→ Node.js를 먼저 설치하세요

### 문제 2: "포트 5000이 이미 사용 중"
→ 다른 프로그램이 포트 5000을 사용하고 있습니다
→ 해결 방법:
```bash
# 포트 5000을 사용하는 프로세스 찾기
lsof -ti:5000

# 프로세스 종료 (PID는 위 명령어 결과)
kill -9 [PID]
```

### 문제 3: "Cannot find module"
→ `npm install`이 제대로 실행되지 않았습니다
→ 다시 실행:
```bash
cd /Users/jisulee/A.홈페이지/server
rm -rf node_modules
npm install
```

### 문제 4: CORS 오류
→ 백엔드 서버가 실행되지 않았습니다
→ 백엔드 서버를 먼저 실행하세요
